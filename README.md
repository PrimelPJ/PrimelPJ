## About

CS student focused on building useful software across the stack — from systems-level Python and TypeScript libraries to mobile apps, fintech platforms, and data pipelines. I like working close to the metal and shipping things that actually get used.

Currently co-founding **Forq**, a food payment platform built on Stripe Issuing, Flinks, Node.js, PostgreSQL, and React Native.
